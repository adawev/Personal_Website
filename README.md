# Personal Portfolio Website

A modern, responsive portfolio website showcasing my projects and skills.

## Tech Stack

- React 19.1.1
- Vite 7.1.7
- React Router DOM 7.9.4
- EmailJS 4.4.1
- Font Awesome 6.4.0

## Setup

```bash
npm install
npm run dev
```
